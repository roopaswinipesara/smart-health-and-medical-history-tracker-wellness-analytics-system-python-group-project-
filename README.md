# smart-health-and-medical-history-tracker-wellness-analytics-system-python-group-project-batch-19
The system securely stores user personal and health-related data in a structured format.
It collects inputs like age, gender, height, weight, BP, heart rate, and blood sugar.
It also includes lifestyle data such as physical activity and sleep duration.
The system analyzes BMI, heart health, diabetes risk, and blood pressure status.
It generates an overall wellness score to summarize the user’s health condition.
It provides health alerts and personalized recommendations for better lifestyle management.
